Glossário de Access Points
SSID
Service Set Identifier — nome da rede Wi-Fi. Pode ser broadcast ou oculto.
WPA3-SAE
Simultaneous Authentication of Equals — substitui PSK, resistente a ataques de dicionário.
RADIUS
Remote Authentication Dial-In User Service — autenticação centralizada para redes corporativas.
Roaming
Transição de cliente entre APs sem perda de conexão. 802.11r acelera o processo.
WPS
Wi-Fi Protected Setup — protocolo com vulnerabilidade grave (PIN brute-force). SEMPRE desativar.
OFDMA
Orthogonal Frequency Division Multiple Access — permite múltiplos clientes simultâneos no Wi-Fi 6.

Modos de Operação do WG302
Access Point
Modo padrão. Conecta clientes Wi-Fi à rede cabeada (infra-estrutura).
Repeater
Repete o sinal de outro AP, estendendo o alcance sem cabo adicional.
Bridge P2P
Liga dois segmentos de rede cabeada via link wireless ponto a ponto.
Bridge P2MP
Um AP central conecta múltiplos APs remotos (hub-and-spoke wireless).

CONECTAR AO AP NO WINDOWS XP
Guia Detalhado
💡 Contexto: O Windows XP usa o Wireless Zero Configuration (WZC) — serviço nativo de gerência Wi-Fi. Ele suporta WEP, WPA-PSK (TKIP) e 802.1X, mas não suporta WPA2 ou WPA3 sem patch. Certifique-se de que o adaptador wireless e seus drivers estão instalados antes de começar.
Método 1 — Conexão pelo Wireless Zero Configuration (WZC)
1
Verificar o serviço WZC ativo
Antes de tudo, confirme que o serviço Wireless Zero Configuration está rodando.
── Verificar via Serviços ────────────────────────
Iniciar → Executar → digitar: services.msc
Localize: "Wireless Zero Configuration" → Status: Iniciado
Se parado: botão direito → Iniciar

── Ou via Prompt de Comando ──────────────────────
C:\> net start "Wireless Zero Configuration"
2
Acessar Conexões de Rede
Abra o gerenciador de conexões de rede do Windows XP.
📂 Iniciar → Painel de Controle → Conexões de Rede
📂 OU: clique com botão direito em "Meu Local de Rede" → Propriedades
📂 OU: Iniciar → Executar → digitar ncpa.cpl → Enter
3
Abrir Propriedades do Adaptador Sem Fio
Na janela "Conexões de Rede", localize o ícone da placa Wi-Fi (geralmente chamado "Conexão de Rede Sem Fio" ou "Wireless Network Connection").
⚠️ Se o ícone não aparecer, o driver do adaptador Wi-Fi não está instalado. Instale o driver correto para sua placa (ex: Intel PRO/Wireless, Broadcom, Atheros).
4
Visualizar Redes Sem Fio Disponíveis
Clique com botão direito no ícone do adaptador Wi-Fi → "Exibir redes sem fio disponíveis".

Patch WPA2 para Windows XP (SP3)
⚠️ Windows XP e WPA2
O Windows XP não suporta WPA2 nativamente. Para usar WPA2-AES é necessário:
① Instalar o Windows XP Service Pack 3 (SP3)
② Instalar o KB893357 (Wireless Client Update para WPA2)
③ Usar driver atualizado do adaptador Wi-Fi com suporte a WPA2
④ No WG302, configurar WPA-PSK/TKIP pois é o máximo suportado sem patches
Resolução de Problemas
❌ Não vejo a rede Wi-Fi
Verifique se o serviço WZC está iniciado. Cheque se o adaptador está habilitado em Gerenciador de Dispositivos. Reinstale o driver.
⚠️ Senha incorreta / falha na autenticação
Confirme o tipo de segurança (WPA vs WEP). Verifique se o SSID cadastrado manualmente está exatamente igual — diferencia maiúsculas.
⚠️ Conectado mas sem internet
Confirme IP correto com ipconfig. Tente ping 8.8.8.8. Verifique se o gateway e DNS estão configurados. Libere/renove IP com ipconfig /release e ipconfig /renew.
🔧 Sinal fraco ou instável
Reduza obstáculos entre o dispositivo e o WG302. Troque o canal do AP (use 1, 6 ou 11). Ajuste a posição das antenas para posição vertical.
