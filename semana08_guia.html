💡 Solução Detalhada do Exercício 2

Explicação: Criação e configuração de zona DNS direta

📝 Comandos Passo a Passo

sudo nano /etc/bind/named.conf.local # zone "example.com" { # type master; # file "/etc/bind/db.example.com"; # }; sudo cp /etc/bind/db.local /etc/bind/db.example.com sudo nano /etc/bind/db.example.com sudo named-checkzone example.com /etc/bind/db.example.com sudo systemctl restart bind9
