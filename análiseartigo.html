CONCEITOS FUNDAMENTAIS
+5 XP
🔄 NAT (Network Address Translation)

Permite acesso à internet dentro do guest. Funciona como um roteador doméstico — agrupa VMs em rede interna, bloqueia acesso direto externo. Usa TCP/UDP sobre IPv4 e IPv6.

🖥️ Host-Only Network

Cria rede entre VMs e o host sem acesso externo. Versão híbrida de Bridge e Internal. Cria interface de loopback virtual no host.

🌉 Bridged Networking

VM aparece como dispositivo físico na rede. Usa driver de filtro para interceptar/injetar pacotes. Ideal para servidores e simulações avançadas.

🔒 Internal Network

Rede visível somente entre VMs selecionadas. Invisível para o host e para o mundo externo. Ideal para ambientes isolados.

🧠 Lembre-se: O VirtualBox disponibiliza até 8 placas PCI Ethernet virtuais por VM. Você pode combinar adaptadores para ter NAT + Host-Only ao mesmo tempo na mesma VM!
