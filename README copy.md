# 📚 Curso Completo: Laboratório de Administração de Sistemas Operacionais de Redes

## 🎯 Visão Geral

Este curso completo contém **20 semanas** de aulas práticas em HTML profissional, organizadas em 4 módulos principais.

---

## 📁 Estrutura dos Arquivos

### Arquivos Principais:
- **index.html** - Página inicial do curso com visão geral
- **plano_ensino_laboratorio_sistemas_operacionais.html** - Plano de ensino oficial
- **semana01.html a semana20.html** - 20 semanas de conteúdo
- **foto_professor.png** - Foto do professor

---

## 🗂️ Módulos do Curso

### 📘 MÓDULO 1: Infraestrutura Física de Redes (Semanas 1-4)
- ✅ **Semana 1**: Introdução e Fundamentos de Cabeamento Estruturado
- ✅ **Semana 2**: Crimpagem de Cabos de Rede (T568A/T568B)
- ✅ **Semana 3**: Patch Panels e Organização de Racks
- ✅ **Semana 4**: Configuração de Switches, Roteadores e Access Points

### 🐧 MÓDULO 2: Linux Server (Semanas 5-11)
- ✅ **Semana 5**: Instalação do Linux Server
- ⚠️ **Semana 6**: Gerenciamento de Usuários e Permissões
- ⚠️ **Semana 7**: Configuração de Rede e DHCP
- ⚠️ **Semana 8**: Servidor DNS com BIND9
- ⚠️ **Semana 9**: Servidor de Arquivos Samba
- ⚠️ **Semana 10**: Shell Script Básico
- ⚠️ **Semana 11**: Scripts Avançados e Cron

### 🪟 MÓDULO 3: Windows Server (Semanas 12-18)
- ⚠️ **Semana 12**: Instalação do Windows Server
- ⚠️ **Semana 13**: Active Directory Domain Services
- ⚠️ **Semana 14**: Usuários e Grupos no AD
- ⚠️ **Semana 15**: Group Policy Objects (GPO)
- ⚠️ **Semana 16**: DHCP e DNS no Windows
- ⚠️ **Semana 17**: File Server, DFS e Print Server
- ⚠️ **Semana 18**: PowerShell para Administração

### 🔍 MÓDULO 4: Finalização (Semanas 19-20)
- ⚠️ **Semana 19**: Monitoramento, Backup e Recuperação
- ⚠️ **Semana 20**: Troubleshooting e Projeto Final
