💡 Solução Detalhada do Exercício 1

Explicação: Script profissional de backup com logging e rotação

📝 Comandos Passo a Passo

nano backup_avancado.sh #!/bin/bash LOG="/var/log/backup.log" exec > >(tee -a $LOG) 2>&1 DATA=$(date '+%Y-%m-%d_%H-%M-%S') ORIGEM="/dados" DESTINO="/backup" DIAS_MANTER=7 echo "[$(date)] Iniciando backup..." tar -czf $DESTINO/backup_$DATA.tar.gz $ORIGEM find $DESTINO -name 'backup_*.tar.gz' -mtime +$DIAS_MANTER -delete echo "[$(date)] Backup concluído!"
💡 DICA: Sempre verifique os resultados de cada comando antes de prosseguir para o próximo passo. Use comandos de verificação como id usuário, groups usuário, ls -la para confirmar as alterações.

Explicação: Monitoramento abrangente do sistema

📝 Comandos Passo a Passo

nano monitor_sistema.sh #!/bin/bash CPU=$(top -bn1 | grep 'Cpu(s)' | awk '{print $2}' | cut -d'%' -f1) MEM=$(free | grep Mem | awk '{print ($3/$2) * 100.0}') DISCO=$(df / | tail -1 | awk '{print $5}' | sed 's/%//') echo "CPU: $CPU% | Memória: $MEM% | Disco: $DISCO%" if (( $(echo "$CPU > 80" | bc -l) )); then echo "ALERTA: CPU alta!" fi
