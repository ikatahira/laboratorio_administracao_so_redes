Glossário de Roteamento
NAT/PAT
Traduz IPs privados para público. PAT mapeia múltiplos hosts numa única porta do IP externo.
ACL
Access Control List — filtra pacotes por origem/destino/protocolo. Política de segurança no roteador.
Router-on-Stick
Uma interface física + subinterfaces dot1Q. Uma subinterface por VLAN, cada uma como gateway.
SSH v2
Secure Shell — acesso remoto seguro ao IOS. Substitui Telnet que transmite em texto claro.
Default Route
0.0.0.0/0 — rota coringa. Todo tráfego sem rota específica é encaminhado aqui (gateway da Internet).
show ip route
Exibe rotas estáticas (S), dinâmicas (O/E) e diretamente conectadas (C) na tabela de roteamento.

Setup de Roteador Básico
📡 Roteadores · ⏱ 30 min · Dificuldade: Médio
+60 XP ▼
Configurar IP nas interfaces, roteamento estático entre duas redes e NAT/PAT.
1

Configure as subinterfaces para inter-VLAN routing (router-on-a-stick):

Router(config)# interface GigabitEthernet 0/0
Router(config-if)# no shutdown

Router(config)# interface GigabitEthernet 0/0.10
Router(config-subif)# encapsulation dot1Q 10
Router(config-subif)# ip address 192.168.10.1 255.255.255.0

Router(config)# interface GigabitEthernet 0/0.20
Router(config-subif)# encapsulation dot1Q 20
Router(config-subif)# ip address 192.168.20.1 255.255.255.0
2

Configure a interface WAN (Gi0/1) com IP público e habilite NAT:

Router(config)# interface GigabitEthernet 0/1
Router(config-if)# ip address 200.10.5.1 255.255.255.0
Router(config-if)# ip nat outside
Router(config-if)# no shutdown

Router(config)# interface GigabitEthernet 0/0.10
Router(config-subif)# ip nat inside

Router(config)# ip nat inside source list 1 interface Gi0/1 overload
Router(config)# access-list 1 permit 192.168.0.0 0.0.255.255
3

Verifique com show ip route, show ip nat translations e faça ping entre VLANs.
