💡 Solução Detalhada do Exercício 1

Explicação: Criação de compartilhamentos com permissões

📝 Comandos Passo a Passo

# PowerShell: Install-WindowsFeature FS-FileServer -IncludeManagementTools New-Item -Path "D:\Compartilhados\TI" -ItemType Directory New-Item -Path "D:\Compartilhados\RH" -ItemType Directory New-SmbShare -Name "TI" -Path "D:\Compartilhados\TI" -FullAccess "EMPRESA\GRP_TI" New-SmbShare -Name "RH" -Path "D:\Compartilhados\RH" -FullAccess "EMPRESA\GRP_RH" $Acl = Get-Acl "D:\Compartilhados\TI" $Ar = New-Object System.Security.AccessControl.FileSystemAccessRule("EMPRESA\GRP_TI", "Modify", "ContainerInherit,ObjectInherit", "None", "Allow") $Acl.SetAccessRule($Ar) Set-Acl "D:\Compartilhados\TI" $Acl
💡 DICA: Sempre verifique os resultados de cada comando antes de prosseguir para o próximo passo. Use comandos de verificação como id usuário, groups usuário, ls -la para confirmar as alterações.

📝 Comandos Passo a Passo

# PowerShell: # Habilitar auditoria de objeto via GPO primeiro $Acl = Get-Acl "D:\Compartilhados\TI" $Ar = New-Object System.Security.AccessControl.FileSystemAuditRule("Everyone", "ReadAndExecute,Modify", "ContainerInherit,ObjectInherit", "None", "Success,Failure") $Acl.AddAuditRule($Ar) Set-Acl "D:\Compartilhados\TI" $Acl # Verificar eventos: Get-WinEvent -LogName Security | Where-Object {$_.Id -eq 4663} | Select-Object -First 10
